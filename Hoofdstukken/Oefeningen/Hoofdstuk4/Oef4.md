# Oefening 4.4: Double gemiddelde

```C#
using System;
					
public class Program
{
	public static void Main()
	{
			// Alle cijfers
			double cijfer3 = 44;
			double cijfer4 = 51;
			
			// Berekening gemiddelde
			double gemiddelde2 = (cijfer3 + cijfer4)/2;
			
			// Weergave gemiddelde
			Console.WriteLine("Het gemiddelde op de examens was" + " " + gemiddelde2 + "%");
	}
}
```
