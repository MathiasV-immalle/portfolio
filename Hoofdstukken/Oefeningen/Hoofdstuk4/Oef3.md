# Oefening 4.3: Integer gemiddelde

```C#
using System;
					
public class Program
{
	public static void Main()
	{
			// Alle cijfers
			int cijfer1 = 44;
			int cijfer2 = 51;
			
			// Berekening gemiddelde
			int gemiddelde = (cijfer1 + cijfer2)/2;
			
			// Weergave gemiddelde
			Console.WriteLine("Het gemiddelde op de examens was" + " " + gemiddelde + "%");
	}
}
```
